# webpack-2-simple-config
Simple configuration for Webpack 2 with Hotloading and Babel

clone the project

```
git clone git@github.com:serdarde/webpack-2-simple-config.git
```

install depencencies
```
npm i
```
// or 
```
yarn install
```

For development 
```
npm run start
```


For production 
```
npm run build
```
